# CloudGuide Lite

A massively simplified static website for cloud learning.

## Stack

- HTML
- CSS
- JavaScript

## What this version does

- Runs directly in the browser
- No Python backend
- No API keys
- No Docker

## Files

- index.html
- style.css
- script.js

## Run with Live Server

1. Open this folder in VS Code.
2. Open index.html.
3. Click Open with Live Server.

## Features

- Simple built-in cloud Q&A
- Mini multiple-choice quiz
- Notes with browser localStorage

## Notes

- This is static mode, so there are no real AI API calls.
- Legacy Flask files may still exist in the repo but are not needed for this website.
